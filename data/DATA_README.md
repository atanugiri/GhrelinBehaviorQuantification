# data/ Directory

This folder contains the behavioral analysis inputs (DeepLabCut-filtered CSVs) used by the project.

## Directory Structure (after setup)

```
data/
├── DlcDataPytorchFiltered/   # DLC CSV outputs (organized by strain and magnification)
│   ├── WhiteAnimals10X/
│   └── WhiteAnimals2X/
└── dlc_table.csv             # Trial metadata (generated by DLCDatabaseSetup)
```

## Data Sources

- **Pose estimation data**: Download from [Harvard Dataverse](https://doi.org/10.7910/DVN/G8CBKJ)
- **Metadata**: `dlc_table.csv` contains trial IDs, experimental conditions, and CSV paths linking to the DLC outputs.

## Setup After Download

Dataverse provides a zip named `DlcDataPytorchFiltered.zip` (large).

On macOS / Linux:

```bash
# Unzip
cd GhrelinBehaviorQuantification/data/
unzip DlcDataPytorchFiltered.zip -d DlcDataPytorchFiltered
rm -rf DlcDataPytorchFiltered.zip
```

On Windows (PowerShell):

```powershell
# Extract with PowerShell
Expand-Archive -Path DlcDataPytorchFiltered.zip -DestinationPath DlcDataPytorchFiltered
Remove-Item DlcDataPytorchFiltered.zip
```

Notes:
- The extracted folder should be named `DlcDataPytorchFiltered` and contain subfolders like `WhiteAnimals10X` and `WhiteAnimals2X` with task-specific subfolders and DLC CSV files.
- The repository's `.gitignore` excludes these large data folders — do not commit them.

See the DLCDatabaseSetup repository for scripts used to generate `dlc_table.csv` and for full metadata extraction instructions: https://github.com/atanugiri/DLCDatabaseSetup/tree/main
